DiodeCtrl documentation.

Description
Use this control to display the value of a pin. This control draws a image of an diode on the screen and within that diode there is a number representing the pin value.
Usage
todo:
Code example on a card:
var using = ["DiodeCtrl"];
var pin1, diode1;

$(function () {
	pin1    = new Pin(16, 999, '57afb108b450ab9c2f820507', 1023);
	//Show the pinvalue 999 as a diode at location x=12 and y=55
	diode1  = new DiodeCtrl(12,55,pin1);
	//make the diode inactive, (shows it as gray) instead of red.
	pin1.active(false);
});
